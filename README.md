# Fragg Force

Fragg Force is a premium luxury perfume e-commerce platform developed for Waqas Shah Perfume, Karachi, Pakistan.

The platform provides a modern online shopping experience for men's, women's, and unisex fragrances with secure authentication, product management, order tracking, invoice generation, WhatsApp integration, and an admin dashboard.

## Features

- Modern Responsive UI/UX
- Product Catalog & Search
- Product Details Page
- Shopping Cart
- Wishlist
- Secure Checkout
- Order Tracking
- PDF Invoice Generation
- WhatsApp Integration
- Email Notifications
- User Dashboard
- Admin Dashboard
- Inventory Management
- SEO Optimized
- Dark Mode Support

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- PostgreSQL
- Prisma ORM
- NextAuth
